# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-17

### Added
- Car dealership smart contract with buy/sell functionality
- Fungible token (CDT) contract for ecosystem
- NFT contract for car ownership certificates
- React frontend with Tailwind CSS
- Token management interface
- NFT minting and gallery
- Comprehensive test suite for all contracts
- GitHub Actions CI/CD pipeline
- Documentation (README, TESTING, DEPLOYMENT, SECURITY)

### Features
- Add cars to inventory (owner only)
- Buy cars with STX tokens
- List/unlist cars for sale
- Mint and transfer CDT tokens
- Create car ownership NFTs
- View NFT gallery
- Wallet integration with Stacks Connect

### Deployed
- Mainnet contract: `SP19PS42C7R7BR4VCX2YN8KPHXSB0ZC19K6PFEKTC.car-dealership`
- Network: Stacks Mainnet (Bitcoin Layer 2)

### Technical
- Clarity smart contracts
- React 18 + Vite
- Tailwind CSS
- @stacks/connect v7
- Full test coverage
